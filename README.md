# FUTURE_DS_02 - Customer Retention & Churn Analysis

## 📊 Project Overview
This project analyzes customer churn using Power BI to understand why customers leave and how to improve retention.

## 🔍 Key Insights
- High churn in monthly contracts
- New customers leave early
- Higher charges increase churn

## 💡 Recommendations
- Promote long-term plans
- Improve onboarding
- Reward loyal customers

## 🛠 Tools Used
- Power BI
- Excel
